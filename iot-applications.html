IOT Applications

Activity 1:

Think about IoT applications:
  • What is an IoT application?
  • Can you think of some Examples?
  • What are the main features needed in an IoT application?
As if you are in the group or as an individual if you are online try to answer these questions by:
  • Create lists.
  • Create a mind map.
  • Create a mini-presentation to explain to others.

Activity 2:

For the scenario of a Smart-Building:
